# わらしべ長者ランナー

日本の昔話「わらしべ長者」をテーマにしたレトロ横スクロール回避ゲームです。

## ゲーム概要

一本のわらから旅を始め、障害物を避けながらアイテムを集めて交換を重ね、最終的に長者を目指します。

## 遊び方

- **ジャンプ**：画面タップ / スペースキー / ↑キー
- 岩・凧・カラス・追い剥ぎを避けながら走り続ける
- ♥ アイテムを取るとアイテムがランクアップ
- 500歩以降は追い剥ぎが出現（近づくと刀を抜く！）

## アイテム交換チェーン

わら → 鶏 → みかん → 反物 → 馬 → 舟 → 屋敷 → 田畑 → こがね

## ステージ

| ステージ | 距離 | 天気 |
|---|---|---|
| 昼の村道 | 0歩〜 | 晴れ |
| 夕暮れの峠 | 700歩〜 | 風 |
| 雨の夜道 | 1500歩〜 | 雨 |

## 障害物

- **岩** ─ 地面に転がる岩。ジャンプで越える
- **凧** ─ 空中の凧。低く走れば当たらない（ジャンプしない）
- **カラス（正面）** ─ 正面から来る。ジャンプで越える
- **カラス（後方）** ─ 後ろから追いかけてくる。⚠ 警告に注意
- **カラス（急降下）** ─ 上から急降下してくる
- **追い剥ぎ** ─ 500歩以降に出現。近づくと刀を構えてくる
